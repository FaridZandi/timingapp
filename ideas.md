# Fun, Low-Effort Things to Build

Constraints:

- Useful with exactly one user
- No community or network effect required
- Interesting on the first run
- Little or no ongoing manual input
- Small enough to prototype without building a platform

## 1. Desktop Archaeologist

Scan a chosen folder and turn its contents into an explorable report:

- forgotten files
- abandoned projects
- duplicate downloads
- unusually old or large files
- amusing filename patterns
- “you created these five things on the same strange evening”

The computer supplies the input. The app makes cleanup feel like discovery.

## 2. Personal Internet Rewind

Read exported browser history and generate a private, local recap:

- topics you fell into this week
- sites you returned to repeatedly
- late-night rabbit holes
- pages you visited but probably never finished
- forgotten pages worth reopening

No new logging habit is required.

## 3. Screenshot Librarian

Watch a screenshot folder, use OCR to extract text, and make everything searchable.

It could automatically identify:

- error messages
- receipts
- conversations
- products
- code snippets
- maps and addresses

This solves a real problem using data you already create.

## 4. “What Changed?” for Your Computer

Take a lightweight daily snapshot of selected folders and summarize meaningful changes:

- projects that became active again
- files that appeared or disappeared
- documents that changed substantially
- bursts of work by time and file type

Think of it as a personal activity digest without surveillance software or manual journaling.

## 5. Accidental Art Finder

Scan your photos and screenshots for visually interesting accidents: odd crops, color palettes, repeated shapes, glitches, reflections, and near-duplicates.

Each run produces a small gallery or collage. The source material already exists, and the output can be genuinely surprising.

## 6. Local File Time Machine

Pick any text file or project folder. The tool quietly saves compact revisions and provides a pleasant visual history of how it evolved.

Unlike Git, it should require no concepts, commands, commits, or messages. It is useful for writing, design files, notes, and experiments.

## 7. Tab Bankruptcy Assistant

Inspect currently open browser tabs and automatically:

- group related tabs
- identify duplicates
- summarize each group
- preserve a searchable session
- suggest the few tabs actually worth keeping

One button turns browser guilt into a clean slate.

## 8. Ambient Sound Postcard

Record ten seconds from the microphone and transform it into a tiny visual artifact using its rhythm, volume, and frequency profile.

It works immediately, needs no account, and gives each place or moment a distinctive “postcard.” Saving or labeling it is optional.

## Best Candidates

1. **Desktop Archaeologist** — strongest mix of surprise, utility, and available input.
2. **Screenshot Librarian** — clearest recurring practical value.
3. **Tab Bankruptcy Assistant** — fastest path to something useful.
4. **Accidental Art Finder** — most playful with almost no user effort.
